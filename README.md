主机管理WEB页面

使用MySQL
   或SQLALchemy
    主机管理(8列)
        ip
    用户表:
        用户名
        密码
    功能:
        1  管理员登陆  用户登陆
        2  管理员后台页面
            -查看所有主机信息(4列)
            -查看详细信息
    
            -增加主机信息(8列)  模态对话框
            -删除主机
    
            -修改主机信息
    
            -添加用户
            -删除用户
    
            -添加分组
            -删除分组
    
            -主机分组操作
            -用户分组操作
    
        3  用户登陆后台页面
            --查看可管理的分组
    
            --查看详细信息
            --修改主机信息



说明:    初始管理员   用户名: admin  密码: admin
